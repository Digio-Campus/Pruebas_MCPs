# Tres en Raya (Tic-Tac-Toe)

Juego clásico de tres en raya implementado en C++ para dos jugadores.

## Compilación

```bash
make
```

## Ejecución

```bash
make run
# o directamente:
./tres_en_raya
```

## Cómo Jugar

1. El juego alterna entre jugador X y jugador O
2. Ingrese la fila (0-2) y columna (0-2) donde desea colocar su marca
3. El primer jugador en conseguir 3 marcas en línea (horizontal, vertical o diagonal) gana
4. Si el tablero se llena sin ganador, es un empate

## Limpieza

```bash
make clean
```
