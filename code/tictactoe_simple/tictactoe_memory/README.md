# Tres en Raya (C++)

Juego de tres en raya implementado en C++ siguiendo las observaciones de diseño de la entidad "TresEnRayaC++Design".

## Estructura
- `TresEnRaya.cpp`: Código fuente principal.
- `Makefile`: Compilación sencilla con flags de advertencia.
- `README.md`: Este archivo.

## Compilación

```sh
make
```

## Ejecución

```sh
./TresEnRaya
```
